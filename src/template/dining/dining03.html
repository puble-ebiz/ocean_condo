속초 횟집

신선하고 싱싱한 활어를 공수하여 제공하는 맛있는 회

속초 횟집 상세안내

속초횟집에는 웅장한 동해가 보입니다.
동해의 신선함이 보이고, 동해의 바다향이 보이고, 동해의 푸르름이 보입니다.
속초횟집은, 청정 동해안에서 직접 잡아 현장에서 즉석으로 준비해드리고 있습니다.
횟감은 청정바다 강원 고성군 현지 어민인 활어회 조리장인이 가진항에서 직접 경매로 낙찰 받아 제공하고 있습니다.

속초횟집 운영시간
  • 금요일, 토요일 운영 (메뉴 사전예약시, 요일과 상관없이 운영합니다.)
  • 18시~22시
속초횟집 판매메뉴
  • 회종류

    모듬회 : 소 30,000원 / 중 50,000 / 대 70,000원

    광어세꼬시 : 소 30,000원 / 중 50,000원

  • 매운탕류

    생우럭매운탕 : 소 30,000원 / 중 50,000원

    생우럭지리 : 소 30,000원 / 중 50,000원

    우럭, 광어 잡뼈 매운탕 : 20,000원

  • 특선메뉴

    회무침 : 30,000원

    물회 : 10,000원 (2인분 부터 주문가능)

  • 식사류

    회덮밥 : 12,000원 (2인분 부터 주문가능)

    공기밥 : 1,500원

속초횟집 이용안내
  • 속초횟집은 오션투유 리조트 실외바베큐장 앞에 있습니다.
  • 금, 토일만 운영하나, 사전예약시 상시 운영될 수 있습니다.
  • 주류 및 음료구매 가능합니다.
  • 속초횟집 전메뉴는 포장 가능합니다.
